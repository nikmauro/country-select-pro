# CountrySelect Pro 🌍



A lightweight, zero-dependency, vanilla JavaScript country selector with localized browser validation and Bootstrap 5 integration.



## Features

- 🚀 **Zero Dependencies**: No jQuery, no heavy libraries.

- ⚡ **Ultra Lightweight**: Less than 4KB minified.

- 🎨 **Bootstrap 5 Ready**: Seamless integration with `input-group`.

- 🔍 **Built-in Search**: Fast filtering of countries and phone codes.

- 🛡️ **Smart Validation**: Uses native browser localized messages.

- 📱 **Responsive & Accessible**: Keyboard navigation and mobile-friendly UI.



## Installation

Just include the JS file from jsDelivr:

```html

<script src="[https://cdn.jsdelivr.net/gh/YOUR_USERNAME/country-select-pro@v4.8/country-select.min.js](https://cdn.jsdelivr.net/gh/YOUR_USERNAME/country-select-pro@v4.8/country-select.min.js)"></script>
