About CountrySelect Pro

CountrySelect Pro is a lightweight and dependency-free country picker built with vanilla JavaScript. It provides fast searching, phone prefix selection and Bootstrap-friendly UI while keeping the bundle extremely small.

  • 🚀 Zero dependencies
  • ⚡ Less than 4KB minified
  • 🔍 Built-in search
  • 📱 Mobile friendly
  • 🎨 Bootstrap compatible
Basic Usage
<input id="country"> <script> new CountrySelect("#country", { defaultCountry: "gr" }); </script>
